About OXTAN

OXTAN is an open-source playground where kids start learning Python the fun way — from the basics, through real-world problems and hands-on messy projects.

Contributors (even total beginners!) help build a tiny command-line language in Python — a silly, powerful, experimental project that grows with the community.

We aren’t here to be perfect. We’re here to be curious, joyful, and helpful. No ads. No paywalls. Just clean, creative chaos. That’s OXTAN.
